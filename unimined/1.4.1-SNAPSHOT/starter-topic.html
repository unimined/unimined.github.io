About Unimined

Welcome to Unimined, a unified Minecraft modding environment that supports both modern and legacy environments. Whether you're working with the latest Minecraft versions or exploring older builds, Unimined provides a comprehensive set of tools to help you build mods with ease.

Features

  • Unified Modding Environment: Supports various Minecraft mod loaders and offers tools for a seamless experience across different modding platforms.

  • Custom Loader Support: Create and integrate your own loaders with the flexibility of Unimined.

Recommended Setup

you can either re-create a unimined build from a standard gradle project or,

  1. take one of the versions from testing

  2. remove includeBuild('../../') from settings.gradle

  3. put a proper version number for the plugin in build.grade
